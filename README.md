## ‚ö° One-Click "Plug & Play" Startup (Windows)

The platform is designed for **Zero Friction**. No manual library installation or path configuration is required.

1.  **Extract/Pull:** Open the project folder in VS Code or your terminal.
2.  **Launch:** Double-click the **`start.bat`** file.
    *   *The system will automatically install all dependencies (Streamlit, Plotly, etc.).*
    *   *It will create your secure `.env` file.*
3.  **Experience (Demo Mode):** Once the launcher opens, select **Option 4 (Run Demo Setup)**.
    *   *This populates the platform with realistic, Geometric Brownian Motion price paths and institutional flows.*
4.  **Visualize:** Select **Option 1 (Start Full Platform)** to open the vibrant Pro Dashboard.
